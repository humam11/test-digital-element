# Тестовое задание

Тестовое задание на стажировку Frontend-разработчик Цифровой Элемент 

## Установка зависимостей

```bash
npm install
```
## Запуск проекта

```bash
npm run dev
```
## Проверка качества кода

```bash
# Проверка ESLint
npm run lint

# Автоматическое исправление
npm run lint:fix
```
